### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/suhailkakar/Codepen-Clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   or 
   
     ```sh
   yarn install
   ```
3. Run the app
   ```sh
   npm start
   ```
